# Uams-php
统一账号管理中心php接口扩展

### Initial setup

1. Install composer. `curl -s http://getcomposer.org/installer | php`
2. Create `composer.json` containing:

    ```js
    {
        "require" : {
            "zhusipu/Uams-php" : "1.0.0"
        }
    }
    ```
3. Run `./composer.phar install`
4. Keep up-to-date: `./composer.phar update`
